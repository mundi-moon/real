주요기능

바로결제 QR코드 / 링크

(1) 바로결제 QR코드 or 링크 클릭

(2) 바로결제 QR코드 or 링크 보관/공유

(3) 구매자(결제하실 고객)에게 전달 *QR코드 이미지 프린트, 링크 카톡 공유 등 상황에 맞게 활용

(4)구매자(결제하실 고객)가 결제페이지 접속 후 금액/결제자정보/카드정보 입력만 하면 결제 완료 *상점정보 확인 필수!

요청 - 일회용주문서

[판매자 매뉴얼]

(1) 금액 / 거래명 입력

(2) 거래방식 선택 (바로결제 or 배송거래)

(3) 1일정산 여부 선택

(4) 일회용 QR코드/링크 발행 후 구매자(결제하실 고객)에게 전달

(TIP) 직거래 시 QR코드 이미지 제시, 배송거래 시 링크 카톡 전송

[구매자 매뉴얼]

구매자(결제하실 고객)가 결제페이지 접속 후 금액/결제자정보/카드정보 입력만 하면 결제 완료

* 상점정보 확인 필수!

요청 - 마켓주문서

[판매자 매뉴얼]

(1) 우측하단의 상품 등록버튼(+) 클릭

(2) 상품 등록 상품명/금액/재고 입력

(3) 거래방식 및 1일정산 여부 선택

(4) 우측상단 [등록] 버튼 클릭 시 상품 등록 완료

(5) 목록에서 해당상품 클릭 후 QR코드 or 링크를 구매자에게 전달

(상품 수정/변경)

(1) 목록에서 해당상품 클릭 후 우측상단 [수정] 클릭

(2) 상품내용(재고/금액 등) 변경, 옵션 삭제, 상품주문서 삭제, 바로결제<->배송거래 변경 등 가능

[구매자 매뉴얼]

결제 페이지 화면

*배송거래 시 배송비 확인 및 주소/전화번호 입력 필수

(1) 옵션 내 구매상품 선택

(2) 결제자정보/카드정보 입력만 하면 결제 완료

영수증 페이지 화면

직접결제

링크, QR코드 전달할 시간도 없이 바쁜 현장에서는, 판매자가 구매자의 카드번호를 직접 입력하여 결제 받으세요.

기기가 없어도 카드 '긁어서 결제'하는 것처럼 신속하고 간편합니다.

(1) 금액입력

(TIP) 자주쓰는 금액 저장

(2) 구매자 카드의 카드번호/유효기간 입력

(3) 결제버튼 클릭하면 끝

정기결제

(1) 정기결제 생성버튼 클릭

(2) 증빙서류 등록 (평균 00분 소요)

(3) 정기결제 계약내용 입력

(계약자명, 계약명, 거래금액, 계약기간, 결제일)

(4) 정기결제자에게 결제요청 링크 전송
